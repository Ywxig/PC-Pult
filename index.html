PC-Pult

Документация:

PC-Pult бот для эмулирования клавиатуры в нутри тлелеграма. Проще говоря вы можите без труда нобирать текст без клавиатуры. какойто осбенной функцией он не обладает просто мне хотелось зделать какойто прикольный бот ну я и решил выполнить это

Бот работает на простой сфеме пользоватеель говорит куда нажать бот нажимает.

/start - запустит бота. бот скажит вам привет и на этом всё

/start

/hot - создаст калвиатуру с горячими клавишами ну я для упровления четыре кнопки для упровления курсором

/hot

/game - создаст калвиатуру с кнопкоми для эмуляции геимпада от xbox360

/game

/kombat - одна кнопка для запуска скрипта борьбы. когда он создовался проверялся он только в игре minecrtaft java

/boor - одна кнопка для запуска скрипта копания. при запуске будет зажиматся кнопка копания для собствнно произведения копания

/kombat

/boor

/ocr - эта комманда используется только в связке с картинкой. ПК при помощи easyOcr прочьтёт вашу картинку и достанет текст на ней после чего вернёт результат обратно в чят.

/ocr - но если вы вы выполненли всё правельно и текста в чяте нет то посмотрите фоил result.txt навернека там будет ваш текст. и главное будте осторожны с русским языком скрипт плохо различяет англиский и кирилиу лудше либо вобще не использовать два языка либо просто после завершения процеса исправте все ошибки пириноса

/html - При использовании просто ввелдите текст рядом с этой коммандой для ввыдо текста с Html разметкой

/html - И введите текст

/elitra - запустит скрипт для автоматическим упровлением подочи феирверка на элитре

/elitra - выбор рижима

/elitra_SET - Введите растояние, и количество фиерверкрв для e-gas
